◆ IntToString()

std::string EN::IntToString ( unsigned int  n)

Function to convert int to string.

One char takes 1 byte, and a int takes 4 bytes. If you try to translate the number 120 into a string, then you will have 3 characters or 3 bytes. This function turns a number into a character, since a character occupies a byte, then you can transfer numbers up to 255 in one byte. This function can be considered as the translation of a number into a number system with a base of 256 This function does not work with negative numbers.

The function gets socket and filename and send file to socket.

Parameters
[in]FileSendSocketthe socket for sending files
[in]FilePaththe full path to the file to be sent
[in]IsStopthe reference to a boolean variable to stop file transfer
[in]ProgressFunctionthe pointer to a function that is used to track the status of file transfer. By default using EN::DownloadStatus
[in]PreviouslySendedSizeThe fifth parameter is needed to continue downloading. Gets the size of the previously transmitted file in bytes. 0 means no previosly sending
[in]ChunksNumberBetweenDelayThe sixth parameter is needed to regulate the file transfer rate. Gets amount of chunks between sending delay. 0 means no delay. The delay between sending the chunks is 20 millimeconds.
Returns
Returns true in case of file transmition success, false otherwise

By default, the library progress function is used to output to the console.
Timeout between sending file chunks set to 20. You adjust the number of chunks that will be sent between the this timeout
