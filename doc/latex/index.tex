I know that no one reads further than the readme, so I want to bring some important things here ~\newline

\begin{DoxyEnumerate}
\item When working with linux, you may notice that std\+::cout does not output characters to the console. Don\textquotesingle{}t worry, your code works as it should, you can check it yourself with cerr. ~\newline

\item On linux, after disconnecting clients from the server, the port may be blocked for a small amount of time. Due to when trying to start the server again, errors will come out. We just have to wait a little ~\newline

\item The U\+DP server cannot send a message to the client if it has not received a message from this client before. This does not apply to unreliable sending of messages in rau classes
\item Note that the maximum size of a udp packet that can be guaranteed to be sent and received is 508 bytes. You can send more, but then the message may not go away ~\newline
